📖 Grading Criteria & Guide

On-Page SEO involves optimizing individual web pages in order to rank higher and earn more relevant traffic in search engines. Here is exactly what our algorithm evaluates:

  • Title Tag: Must exist and ideally be between 50-60 characters so it doesn't get truncated in Google Search Results.
  • Meta Description: Should exist and vividly describe the page within 120-160 characters.
  • H1 Heading: A page should have exactly *one* H1 element summarizing the core topic.
  • H2 Headings: Confirms the presence of subheadings indicating a well-structured document.
  • Image Alt Attributes: Every `<img>` tag must have an `alt` attribute for accessibility and Google Image indexing.
  • Canonical Link: Prevents duplicate content issues by explicitly telling search engines which URL is the master version.
Pro Tip: Missing image alt tags and multiple H1 tags are the top two completely avoidable SEO errors that developers make. Fixing these provides easy ranking wins!
