📖 How it Works & Guide

Curious about what makes your favorite websites tick? This tool scans the source code for signatures of popular web technologies.

  1. Copy the Source: Go to the website you want to analyze, right-click, and select "View Page Source".
  2. Paste & Scan: Paste the entire HTML content into the input box above and click "Scan Tech Stack".
  3. Review Results: Our engine looks for script patterns, meta tags, and class names to identify frameworks like React, Next.js, and various CDNs or analytics tools.

Learn more: MDN Web Docs | Google Web.dev

Pro Tip: For developers, using client-side tools like this one prevents sensitive data (API keys, code snippets) from being logged on external servers, providing an essential layer of security for your workflow.

Tip: For more accurate results, ensure you include the <head> section of the HTML.
