❓ Frequently Asked Questions
How accurate is this stopwatch?

This stopwatch uses requestAnimationFrame with performance.now() for the highest accuracy available in a browser — typically within 1–2 milliseconds, which exceeds the precision of most phone apps and is sufficient for sports training.

Does it work if I switch tabs?

Yes. The stopwatch uses system timestamps (performance.now()) rather than interval-based counting, so it remains perfectly accurate even when the tab is in the background. The display updates when you return.

What does the Lap button do?

The Lap button records the time for the current segment while the stopwatch continues running. It shows both the individual lap time and cumulative total time — useful for tracking intervals in running, swimming, or any repetitive activity.

Can I use this for workout intervals?

Absolutely! Use the Lap button at the end of each work/rest period. The lap table shows individual segment times and cumulative totals, making it perfect for tracking Tabata, HIIT, sprint intervals, or swimming laps.

Is this as accurate as a physical stopwatch?

For most purposes, yes. Browser stopwatches using performance.now() achieve 1–2ms accuracy, matching most consumer digital stopwatches. Only professional timing systems (Olympic-grade) are more precise at sub-millisecond levels.
